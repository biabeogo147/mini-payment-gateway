# Getting Started

Use this folder for first-run and repeatable developer/operator commands.

- `local-setup.md` - shortest path to install dependencies, migrate DB, and
  start the API.
- `runbook.md` - full verification, smoke, and demo operating guide.
- `e2e-payment-demo.md` - complete visual VietQR journey using the local demo
  merchant checkout, provider simulator, worker, and both dashboards.
- `vietqr-pilot-demo.md` - API/Postman-focused VietQR pilot walkthrough.
