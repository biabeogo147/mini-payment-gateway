Routes & \code{GET /v1/internal/users}, \code{POST /v1/internal/users}, \code{PATCH /v1/internal/users/{user_id}}, \code{POST /v1/internal/users/{user_id}/reset-password}. \\
Actor and auth & Internal \code{ADMIN} session only. \\
Key request fields & Create and update carry internal user identity fields; reset-password has no body. \\
Key response fields & User identity, role, status, timestamps, and one-time generated password on create or reset. \\
Important business rules & Internal user administration is Admin-only; generated passwords are returned once and not retrievable later. \\
Important error cases & Auth and RBAC failures, validation errors, internal user lookup failures. \\
Related scenarios & Ops Dashboard internal user management flows. \\
