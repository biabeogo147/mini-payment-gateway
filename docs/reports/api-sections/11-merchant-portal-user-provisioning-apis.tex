\chapter{Merchant Portal User Provisioning APIs}

Merchant portal user management belongs to the Ops surface and is
\apiid{ADMIN}-only. \apiid{OPS} users can continue operational workflows but
cannot create merchant portal users, change portal roles or status, or reset
portal passwords.
