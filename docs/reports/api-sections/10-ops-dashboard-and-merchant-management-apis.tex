\chapter{Ops Dashboard And Merchant Management APIs}

Ops APIs are internal-only. They are used by authenticated \apiid{ADMIN} and
\apiid{OPS} users. Routes that change sensitive state require an audit reason.
Controller logic normalizes actor role and actor id from the internal session;
client-supplied actor identity is not trusted.
